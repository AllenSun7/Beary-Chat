## Run the Python with screen
* keep the python script running in the backgroup:
    - use screen: 
    ```
    screen
    ```
    - run script: 
    ```
    python3 main.py &
    ```
* kill process
    - list of process: 
    ```
    ps
    ```
    - find specific process: 
    ```
    ps ax | grep main.py
    ```
    - list of python process
    ```
    ps ax | grep python
    ```
    - kill the process, "process number" 
    ```
    kill "process"
    ```
